# Personal Portfolio

This project was created as part of my **Day 1 internship assignment**, where I was introduced to the fundamentals of HTML and CSS.

The project focused on learning basic web development concepts by building a personal portfolio website. It helped me understand page structure, styling, layouts, responsive design, and organizing content using HTML and CSS.

The website includes multiple pages such as Home, Projects, Skills, and Contact to showcase my work and technical skills.

## Technologies Used

* HTML5
* CSS3

## Features

* Home Page
* Projects Page
* Skills Page
* Contact Page
* Responsive Design
* Clean User Interface

## Project Structure

```text
├── index.html
├── projects.html
├── skills.html
├── contacts.html
├── style.css
└── assets/
```

## Running the Project

Open `index.html` in any web browser to view the website.
